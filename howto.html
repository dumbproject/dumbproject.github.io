how i made this website

  1. create/have a github account
  2. install git
  3. make a folder for this project, open the folder in cmd/terminal, and type
    git init
  4. make a text file and name it index.html

so every .html file needs <!doctype html>
so the computer knows what kind of file it is
then, make opening and closing <html> tags
and you put <html lang="en">

then you put opening and closing <head> and <body> tags inside the <html> tag
and inside the head you put a <title>title for your page</title>
and also <style>css styles</style> to make your page look fancy
<!doctype html>
<head>
  <title>example title</title>
</head>
<body>
  here is me saying things in my example html page
</body>
</html>
