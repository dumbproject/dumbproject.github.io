how i made this website

  1. create a github account
  2. install git
  3. make a folder for this project, open the folder in cmd/terminal, and type git init
  4. make a text file, save it as index.html
  5. open the project folder in terminal/cmd
  6. type git status (optional, helpful)
  7. git add the files you create
  8. git commit the changes you made
  9. git push your updated version to github
  10. wait a few minutes for the changes to take effect and browse your own site at sitename.github.io

every .html file starts with <!doctype html> at the top
this tells the computer what kind of file it is
make a closing <html> tag at the bottom

put an opening and closing <head> tag in between the two
for info about the page
and <body> tags under the <head> tags, before the closing html tag
and inside the head you put a <title>title for your page</title>
and also <style>css styles</style> to make your page look fancy
