how to make a free website through github

  1. create a github account
    2. (name it what you want your site named)
  2. install git
  3. make a folder for this project
    open the folder in the command line
    and type git init
  4. make a text file, save it as index.html
  5. open the project folder in terminal/cmd
  6. type git status (optional, helpful)
  7. git add the files you create
  8. git commit the changes you made
  9. git push your updated version to github
  10. wait a few minutes for the changes to take effect
    and browse your own site at sitename.github.io




how to make an html file

the basics

  • html files are text files saved as name.html and viewed in a browser as a web page
  • html uses "elements" that tell a web browser how to structure the page
    • elements let you style or alter the contents of your page
    • elements use an opening <tag> and closing </tag>
    • content inside the tags are affected by what element is used
  • html files use css files ("stylesheets") and javascript files ("scripts")
    • css is for changing colors or fonts or structuring the layout of the page
    • javascript is a programming language useful for many things
    • css or javascript is not required for a basic html page

required for every html file

  • all html files start with <!DOCTYPE html> at the very top and end with a closing </html> tag at the very bottom
    • (this tells the computer what kind of file it is)
  • opening <head> and closing </head> tags between the html tags
    • this is where you put your <title>website name</title> or other important details
    • you can also put <style>css</style> inside style tags to add design to your site (i'll explain css later)
  • and <body> tags under the <head> tags, before the closing html tag

basic example html file

<!DOCTYPE html>
<head>
  <title>example title</title>
</head>
<body>
  here is me saying things in my example html page

</html>

how to make a javascript file

javascript is less easy but still really really easy
javascript is a high-level programming language, which means that it's easier to learn and more "abstract" than low-level languages (like C)
javascript, like most (all?) programming languages, has just a few different rules you need to know
you use "statements" to tell it what to do, like if-then (or if-then-else)
it also uses "variables" which store data for the statements to use

in a simplified sense, it works like this:

var = food

if (i am hungry) { eat food } else (if i am not hungry) { keep the food for later }
